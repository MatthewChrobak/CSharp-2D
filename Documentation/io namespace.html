IO Namespace

The IO namespace contains a collection of classes designed to handle game files and other uniquely structured byte arrays.

Namespace Structure

- Classes

Name Description
public static class Compression Compresses and extracts .zip files and directories.
public class DataBuffer Reads and writes values dynamically to a stream.
public static class FolderSystem Handles creating necessary folders.
public static class Serialization Serializes and deserializes .xml files.
