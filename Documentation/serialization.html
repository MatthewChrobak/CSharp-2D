HEADER

This class is used for serializing objects to .xml files, and deserializing .xml files.

Class Structure

- Methods

Name Description
public static T Deserialize<T>(string, Type) Returns a deserialized .xml file of a specified type. Throws an exception if the file does not exist.
public static void Serialize<T>(string, Type, object) Serializes an object of the type specified into a .xml of a path specified. The files is overwritten if it already exists.
