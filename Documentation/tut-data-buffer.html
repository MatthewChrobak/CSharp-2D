How to use the DataBuffer class

The DataBuffer class is a very useful class to use; especially when working with packet sending. Conceptually, it's not very complicated. Create a new instance of the class, and either read or write values to or from it.

- Writing to the DataBuffer

				

// Create a new object.
var buffer = new DataBuffer();

// Write values to it.
buffer.Write("String value");
buffer.Write(true);
buffer.Write(1.55f);
buffer.Write(new byte[] { 1, 2, 3, 4, 5 });

- Reading from the DataBuffer

				

// Suppose 'array' is a byte array containing information we wish to get. 
var buffer = new DataBuffer(array);

// Read the values from the object.
string name = buffer.ReadString();
int number = buffer.ReadInt();
byte[] data = buffer.ReadBytes();
