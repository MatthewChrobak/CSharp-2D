How to serialize and deserialize an object

All methods in the Serialization class are static, and can be referenced through the classname.



- Serialization

				

// Create an instance of a player class.
var player = new Player();

// Deserialize the player into the file 'player.xml'.
Serialization.Serialize<Player>(Game.DataPath + "player.xml", player.GetType(), player);

- Deserialization

				

// Create an instance of a player class.
var guy = new Player();
				
// Set 'guy' to be the deserialized player data in the file 'player.xml'.			
guy = Serialization.Deserialize<(Player>(Game.DataPath + "player.xml", player.GetType());
