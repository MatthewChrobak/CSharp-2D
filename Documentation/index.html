CSharp 2D Documentation

CSharp 2D is a project framework written in C# by Matthew Chrobak. The framework may be used to develop singleplayer and multiplayer two dimentional video games.

- Templates

There are three project templates currently available for use. These templates are separated into two categories: Multiplayer, and Singleplayer.


The singleplayer templates consist of only one item: the singleplayer engine template. This template contains the Audio, Data, IO, and Graphics namespaces which provide core functionality for most basic singleplayer game needs.

The multiplayer templates consist of two items: the server and client templates. The client template contains the same namespaces as the singleplayer engine template, with the added networking namespace. Applications made with the client template function as the visible frontend of your game, which your users require in order to play.

The server template contains only the Data, IO, and Networking namespaces. Applications made with this template function as the core of the game, and send relevant data to the connected clients.

To learn more about each namespace, and how to begin to build on your projects with these templates, select a namespace on your left-hand side.
