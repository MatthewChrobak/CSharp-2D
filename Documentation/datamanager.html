Data Manager

This class handles the loading, saving, and storing of game data. This class should also contain references to data for the rest of the applciation to use.

Class Structure

- Methods

Name Description
public static void Load() Called when the application starts. Loads all data relevant to the application on startup.
public static void Save() Called right before the application closes. Saves all data relevant to the application.
