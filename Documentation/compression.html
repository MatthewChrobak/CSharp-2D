Compression

This class uses GZip to compress and decompress files or directories in .zip files.

Class Structure

- Methods

Name Description
public static void CompressDirectory(string, string) Compresses the specified directory to a .zip file of the specified path. Throws an exception if the directory does not exist.
public static void DecompressDirectory(string, string) Decompresses a .zip file containing a directory to the specified path.
public static void CompressFile(string) Compresses the specified file to a .zip file. Throws an exception if the file does not exist.
public static void DecompressFile(string, bool) Decompresses the specified .zip file containing a file. Throws an exception if the file does not exist. Deletes the file afterwards if specified.
