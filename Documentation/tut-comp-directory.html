How to compress and decompress a directory

The CompressDirectory method is meant to be used if you wish to compress multiple files into a .zip file. All methods in the Compression class are static, and can be referenced through the classname.

If you wish to compress or decompress a single file in a .zip file, you need to use the CompressFile and DecompressFile methods.

Use these methods for handling multiple files only.



- Compression

				

// Let 'path' be the location of the directory we wish to compress.
string path = Game.StartupPath + "data\\";

// Let 'zipPath' be the location of the zip file we wish to make.
// Make sure the save path is not contained in the source directory.
string zipPath = Game.StartupPath + "file.zip";

// Compress the directory 'path' into a file 'zipPath'.
Compression.CompressDirectory(path, zipPath);

- Decompression

				

// Let 'zipPath' be the location of a .zip file made with the CompressDirectory method.
string zipPath = Game.StartupPath + "file.zip";

// Let 'sourcePath' be the directory we wish to put the decompressed files into.
string sourcePath = Game.StartupPath + "data\\";
Compression.DecompressDirectory(path);

// Decompress the contents of 'zipPath' into 'sourcePath'.
Compression.DecompressDirectory(zipPath, sourcePath);
