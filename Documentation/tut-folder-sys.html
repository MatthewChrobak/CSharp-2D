How to add a necessary folder

When creating a game, you might, for example, need to have a folder for placing error log files. Ensuring that such a folder exists is very simple and can be done by using the FolderSystem class found in the Game.IO namespace.


The FolderSystem is a static class that has only one method called Check. Inside this method are two components: a string array, and a foreach loop. The array contains directory paths, and the foreach loop goes through each member in the array and creates the path if it does not exist.

Adding a new path is very simple. 

				

// Put all paths to check here separated by a comma.
string[] folders = {
	// Paths can either be contained in classes, or composed in the array itself.
	Game.StartupPath + "\\data\\npcs\\",
	DataManager.NPCDataPath
};

The foreach loop at the end will take care of the rest.
