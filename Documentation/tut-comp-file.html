How to compress and decompress a file

The CompressFile method is meant to be used if you wish to compress a single file into a .zip file. All methods in the Compression class are static, and can be referenced through the classname.

If you wish to compress or decompress multiple files in a .zip file, you need to use the CompressDirectory and DecompressDirectory methods.

Use these methods for handling single files only.



- Compression

				

// Let 'path' be the location of the file we wish to compress.
string path = Game.StartupPath + "file.txt";

// Compress the file into a .zip file called "file.txt.zip".
Compression.CompressFile(path);

- Decompression

				

// Let 'path' be the location of a .zip file made with the CompressFile method.
string path = Game.StartupPath + "file.txt.zip"
Compression.DecompressFile(path);

// If you want the .zip file to be deleted after decompression.
Compression.DecompressFile(path, true);
