DataBuffer

This class is generaly used for dynamic reading and writing to a byte array with specific ordering.

Class Structure

- Constructors

Name Description
public DataBuffer() Objects are initialized for writing to memory.
public DataBuffer(string) If the file exists, the bytes of the file are read into memory, and the reader is initialized. Otherwise, throw an exception.
public DataBuffer(byte) Objects are initialized for reading from memory containing the bytes specified.
~DataBuffer() Disposes of any remaining objects.

- Fields

Name Description
private MemoryStream _buffer The System.IO object containing the memory to be either written to, or read from.
private BinaryReader _reader The System.IO object that reads from the memorystream.
private BinaryWriter _writer The System.IO object that writes to the memorystream.

- Methods

Name Description
private void Dispose() Disposes of all initialized class fields.
public void Write(string) Writes a string to the memorystream.
public void Write(char) Writes a char to the memorystream.
public void Write(bool) Writes a boolean to the memorystream.
public void Write(byte) Writes a byte to the memorystream.
public void Write(short) Writes a short to the memorystream.
public void Write(int) Writes a int to the memorystream.
public void Write(long) Writes a long to the memorystream.
public void Write(byte[]) Writes the length of the byte array, and then the byte array to the memorystream.
public void Write(float) Writes a float to the memorystream.
public void Write(double) Writes a double to the memorystream.
public void Write(double) Writes a double to the memorystream.
public void Write(Networking.Packet) Writes a packet enumeration value casted to an integer to the memorystream.
public string ReadString() Reads a string from the memorystream. Disposes of the object if the end is reached. Returns an empty string if unable to read from the memorystream.
public char ReadCbar() Reads a char from the memorystream. Disposes of the object if the end is reached. Returns an empty char if unable to read from the memorystream.
public bool ReadBool() Reads a boolean from the memorystream. Disposes of the object if the end is reached. Returns an false if unable to read from the memorystream.
public byte ReadByte() Reads a byte from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public short ReadShort() Reads a short from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public int ReadInt() Reads a int from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public long ReadLong() Reads a long from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public float ReadFloat() Reads a float from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public double ReadDouble() Reads a string from the memorystream. Disposes of the object if the end is reached. Returns 0 if unable to read from the memorystream.
public byte[] ReadBytes() Reads an integer specifying length, and reads a byte array of that length from the memorystream. Disposes of the object if the end is reached. Returns an empty byte array if unable to read from the memorystream.
public void Save(string, bool = true) Saves a file containing the bytes in the memorystream of the specified path. If the file already exists, delete it.
public byte[] ToArray(bool = true) Converts the memorystream into a byte array. Disposes of the object if specified. Returns an empty byte array if unable to read from the memorystream.
public byte[] ToArray(bool = true) Converts the memorystream into a byte array. Disposes of the object if specified. Returns an empty byte array if unable to read from the memorystream.
public byte[] ToPaddedArray() Returns the memorystream converted to a byte array, but starting with the length of the byte array. The object is disposed of prior to returning.
